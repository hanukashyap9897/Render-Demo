This project predicts whether a student will get placed or not based on their academic and personal details.  
I trained a machine learning model and deployed it using Render.

---

 Project Files

- *model.pkl* → Trained ML model  
- *requirements.txt* → Python libraries  
- *app.py* → Flask app for deployment  
- *templates/* → HTML files (if any)

---

 Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-Learn  
- Flask  
- Render (Deployment)

---

 What the Model Does

The model takes student data like:
- CGPA  
- IQ  
- Gender  
- Stream  
- Profile Score  

And predicts:
- *Placed (1)*  
- *Not Placed (0)*
